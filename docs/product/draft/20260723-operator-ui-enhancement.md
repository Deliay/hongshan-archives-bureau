# 干员界面优化需求

## 背景

当前干员档案、干员种族、干员阵营三个页面的干员展示采用横向卡片布局（小头像 + 文字信息），视觉层次不够突出，干员头像作为核心辨识元素展示面积偏小。

## 目标

1. 统一三个页面的干员展示形式，产出通用竖向卡片组件
2. 突出干员头像，采用竖向长方形布局（高 > 宽），比例 152 × 212
3. 保留稀有度标识和名称展示，复用现有稀有度色条能力

## 用户故事

- 作为管理员，我希望在浏览干员档案时，能通过大头像快速辨认干员
- 作为管理员，我希望在查看种族/阵营成员时，能看到统一的卡片样式
- 作为管理员，我希望卡片底部有稀有度色条和名称，便于区分不同稀有度的干员

## 产品规格

### 卡片样式

```
┌───────────────────────┐
│ [职业图标]             │ ← 左上角可选徽章
│                       │
│     干员头像           │ ← 核心展示区域
│     (portrait)        │
│                       │
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│ ← 稀有度渐变遮罩
│        干员名称        │ ← 底部名称
│▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓│ ← 稀有度色条
└───────────────────────┘
```

- **宽高比**: 152:212（约 71.7%）
- **交互**: 整张卡片可点击，跳转至干员详情页
- **悬停**: 边框高亮（金色）

### 适用页面

| 页面 | 卡片尺寸 | 说明 |
|------|----------|------|
| 干员档案 | 中等 (114×159) | 保留筛选/排序/分组功能 |
| 干员种族 | 小 (76×106) | 种族分组下成员展示 |
| 干员阵营 | 小 (76×106) | 阵营分组下成员展示 |

## 验收标准

1. 三个页面的干员卡片统一为竖向长方形样式
2. 卡片比例为 152:212
3. 稀有度色条正确显示
4. 干员名称在卡片底部展示
5. 点击卡片跳转至干员详情页
6. 响应式布局正常（移动端/平板/桌面）
