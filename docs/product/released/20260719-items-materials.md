# 道具材料

道具材料模块收录塔卫二上的物资、材料、消耗品与收集品，提供列表浏览与快速提示。

## 物品列表

列表页以物品面板网格展示所有物品，每个面板显示：

- 物品图标
- 名称
- 稀有度色条
- 数量（如适用）

### 搜索与筛选

| 能力 | 说明 |
|------|------|
| 搜索 | 按名称或 ID 模糊搜索 |
| 物品类型 | 材料、消耗品、收集品、礼物等 |
| 稀有度 | 按星级过滤 |
| 显示类型 | 按游戏内展示分类过滤 |
| 贵重标签 | 按贵重仓库分类过滤 |

### 排序、分组与分页

- 排序：显示类型、稀有度、物品类型，支持升序/降序。
- 分组：按显示类型、物品类型或贵重标签分组。
- 分页：每页 24 / 48 / 96 / 全部。

## 物品提示

点击物品可弹出提示浮层，展示：

- 物品大图与名称
- 稀有度
- 类型
- 描述文本
- 装饰性说明（背景故事或Flavor文本）
- 获取途径
- 宝箱内容（如为宝箱类道具）

## 物品种类

| 种类 | 说明 |
|------|------|
| 材料 | 角色突破、武器突破、技能升级材料 |
| 消耗品 | 药剂、食物等 |
| 收集品 | 地图拾取的档案/收集物 |
| 礼物 | 好感度赠送物品 |
| 武器 | 已归入武器档案 |
| 装备 | 已归入装备系统（待建设） |
| 种子/种植 | 工厂种植系统相关（待建设） |

## 关联入口

- 干员卷宗、武器卷宗、敌人卷宗中的材料均可点击弹出道具提示。
- 后续将建设物品独立卷宗页，展示完整获取途径与用途反查。
